## Quickstart

The following code snippet demonstrates a simple app that uses the React UI Crud [InputField](/docs/components/InputField/)
component:

```jsx
import * as React from "react";
import { Form, TextField, SubmitButton } from "react-ui-crud";

export default function App() {
  return (
    <div className="App w-96">
      <Form onSubmit={console.log}>
        <TextField name="firstName" label="First Name" />
        <TextField name="lastName" label="Last Name" />
        <SubmitButton label="Click to submit" />
      </Form>
    </div>
  );
}
```

You can play around with this code in the interactive Code Sandbox demo below.
